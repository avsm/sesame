Collections

Collections are at the very heart of Sesame and was the first thing Sesame could do. A Collection is simply a set of files that share the same metadata structure. This is expressed in the "Jekyll Format", a header (frontmatter) of yaml followed by markdown.

Once you have specified the shape of the metadata using types, Sesame can more or less handle the rest until you want to customise the output which you most likely will!
